Guess the Song

  • Commands:
  • /playlist spotify-link
  • /turns number
  • /startGame

Commands can be executed only by the lobby admin (the first player who joins a lobby)

  • Guess Requirements:
  • You have to guess the title and all the artists in order to skip the song, if not skipped it will last 30 seconds
  • Title must be in a different message than artists
  • Artists can appear in the same message or in different messages
  • The song title are the same as in Spotify, but without everything inside (), [], or everything after a - if it contains the words "remastered", "remix", "radio edit" and similar.
