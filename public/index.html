Guess The Song

A Spotify-based song-guessing multiplayer game

Join a Room

How to play

I. Joining a Room

Choose an username and type the name of the room you want to join. If you want to create a room, you can type anything you like.

II. Setting up the game

The first user to join a room gets admin privileges and can access the settings by clicking the small gear icon in the top-right corner. In the settings page the admin can paste a Spotify Playlist's link (the playlist have to be public) and load the playlist. The page will then display the number of tracks that can be played from that playlist (the game uses Spotify's previews, which are not available for every song), and the admin can decide how many of them will be played.

III. Playing

Once the game starts, the players have 30 seconds to guess the title of the songs and all the artists involved. The title have to be written separatedly from the artists, while the artists can be written together in a single guess. Players get a point for every correct guess. If the title and all the artists are guessed before the end of the 30 seconds, the song automatically stops.
